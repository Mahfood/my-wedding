## My personal wedding application
### Application information
  * ruby 2.3.1
  * rails 5.0.0
  * Postgresql database

### How to run the application.
1. bundle install
2. bundle exec rake db:create
3. bundle exec rake db:migrate
4. rails server
Rails server starts **puma**
